# 실행 방법 (터미널)

프로젝트 루트: `d:\Cliva RC\Promotion`

---

## 0. 사전 확인

```powershell
node -v    # v22+
npm -v
```

---

## 1. 최초 설정 (1회)

```powershell
npm install
copy server\.env.example server\.env
copy client\.env.example client\.env
npm run db:migrate
```

`server\.env` · `client\.env`에서 **【입력】** 표시 항목을 채웁니다. (항목 설명은 각 `.env.example` 참고)

홍보 영상 반영 (client\.env 수정 후):

```powershell
npm run build
```

---

## 2. 개발 실행 (행사 PC · 로컬)

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette"
npm run dev
```

| 용도 | URL |
|------|-----|
| 부스 화면 | http://localhost:5180 |
| API | http://localhost:4000 |
| 헬스체크 | http://localhost:4000/api/health |

종료: 터미널 `Ctrl+C`

---

## 3. 운영 실행 (PC 1대 · 빌드 후)

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette"
npm run build
cd server
npm start
```

접속: http://localhost:4000

종료: 터미널 `Ctrl+C`

---

## 4. HTML 프로토타입 (서버 없음)

```powershell
cd "d:\Cliva RC\Promotion"
start booth_roulette.html
```

---

## 5. 행사 당일

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette"
npm run dev
# 또는 운영: npm run build && cd server && npm start
```

브라우저 → http://localhost:5180 (개발) 또는 http://localhost:4000 (운영)

---

## 6. 자주 쓰는 명령

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette"

npm run dev              # 개발 (API 4000 + 웹 5180)
npm run build            # 프로덕션 빌드
npm run db:migrate       # DB 스키마 적용
npm run db:seed          # 기본 재고 시드
```

API만:

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette\server"
npm run dev              # 개발 (tsx watch)
npm start                # 운영
```

클라이언트만:

```powershell
cd "d:\Cliva RC\Promotion\booth-roulette\client"
npm run dev              # :5180
```

## 8. 문제 해결

**DB locked**

```powershell
Get-Process node -ErrorAction SilentlyContinue | Stop-Process -Force
cd "d:\Cliva RC\Promotion\booth-roulette\server\prisma"
Remove-Item dev.db, dev.db-journal -ErrorAction SilentlyContinue
cd ..\..
npm run db:migrate
```

**API 응답 확인**

```powershell
Invoke-RestMethod http://127.0.0.1:4000/api/health
```

**포트 사용 확인**

```powershell
netstat -ano | findstr :4000
netstat -ano | findstr :5180
```
